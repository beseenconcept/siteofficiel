Comment convevons nous nos applications ?

Prennons cet exemple !

Notre client désire une application pour gérer les achats et les ventes de ses produits.
Il souhaite aussi gérer ses prospects, clients et fournisseurs, suivre toutes les commandes ainsi que les stocks d'articles par magasins pour ses trois points de vente. Enfin les informations devront être saisies via formulaire.

Nous allons donc réalisé, pour notre client, une application "web" hébergée qui sera accessible par un simple navigateur internet depuis tous les points de ventes. Les informations seront stockées dans une base de données unique facilitant ainsi les sauvegardes. Enfin nous développeront une interface utilisateur sécurisée permettant aux employés et à notre client de saisir ou manipuler les informations par la biais de formulaires.
